# 🏝️ Koh Samui Trip

An interactive itinerary for our 9-day Thailand trip — **26 June → 5 July 2026**, 8 friends.

**Route:** Phuket → Don Sak → Koh Samui → Koh Phangan (Black Moon Party) → back to Koh Samui → Phuket → home.

## What's inside

- **Tappable day-by-day timeline** with check-in times, confirmation codes, one-tap calls to each host, and map links.
- **"What to wear"** suggestion on every day, tuned to that day's plan.
- **Travel Wardrobe checklist** — quantities for the full trip (~50 pieces), with tick-off boxes and a progress bar. Each person's ticks save automatically on their own device.
- **Pack-once essentials** card and a heads-up on the one tight travel connection (Sat 27 June).

It's a single static `index.html` file — no build step, no dependencies.

## Hosting (Vercel)

This deploys on Vercel's free Hobby plan with zero configuration:

1. Push this repo to GitHub.
2. On [vercel.com](https://vercel.com), **Add New → Project → Import** this repo.
3. Leave all settings default (Framework Preset: *Other*) and **Deploy**.

You'll get a live URL like `koh-samui.vercel.app`.

## Updating

Edit `index.html` (or re-upload it) on GitHub — Vercel redeploys automatically within seconds. No commands needed.

## Notes

- Packing ticks are stored per device/browser, so everyone tracks their own packing privately. Opening the link on a different device starts fresh.
- All times are local Thailand time (ICT). Bookings are under Ruthvika.
